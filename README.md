# TownNode

## Description

Bots sign node self hosted.

## Deploy
* Make sure your device has Docker installed.
* Download [Deploy.zip](https://github.com/CalvinLeeC7E/TownNode/blob/main/TownNodeDeploy.zip) to your device.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
